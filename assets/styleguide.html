Built for designing.

Welcome to your custom build of Tachyons.

Create fast loading, highly readable, and 100% responsive interfaces with as little css as possible.

Getting Started

Copy the link style tag and paste it it in the head of the html file(s) you want to include this custom tachyons build in 

<html lang="en">
  <title> </title>
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://tachyons.pub/0.23.0/f26a3e47c572ec1d9c47503e8ef77145/css/tachyons.min.css" />
  <body>
  </body>
</html>

  Measures:
    measure = literal value 30em (base line length)
    measure-narrow = literal value 34em (narrow line length)
    measure-wide = literal value 20em (wide line length)

  Media Query Extensions:
    -m  = 48em
    -l  = 64em

*/

}
